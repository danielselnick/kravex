# Contributing to Kravex

We welcome contributions! By submitting a pull request, you agree to the following:

## Developer Certificate of Origin (DCO)

All contributions must include a `Signed-off-by` line in the commit message,
certifying that you have the right to submit the work under the project's current terms.

```
Signed-off-by: Your Name <your.email@example.com>
```

Use `git commit -s` to add this automatically.

## What This Means

- You certify that you wrote the contribution, or have the right to submit it
- You understand that your contribution is public and may be redistributed

## Code Style

See [CLAUDE.md](CLAUDE.md) for project conventions, comedy laws, and the full style guide.

## Enterprise Directory

The `/ee/` directory is proprietary. Community contributions to `/ee/` are not accepted
without a separate Contributor Agreement (CLA). Contact daniel@kravex.net.
